# 🚦 Traffic Data Analysis

A Python-based Traffic Data Analysis System developed to process CSV traffic datasets and visualize hourly vehicle frequencies through interactive histogram charts. The application enables traffic pattern comparison between multiple road junctions using a graphical Tkinter interface.

---

## 📌 Overview

This project analyzes traffic flow data collected from different road junctions and generates visual histogram representations of vehicle frequencies for each hour of the day. The system supports multiple datasets and provides date-based traffic analysis with input validation and error handling.

---

## ✨ Features

- 📂 Process multiple CSV traffic datasets
- 📊 Analyze hourly vehicle frequencies
- 📈 Generate histogram visualizations
- 🚗 Compare traffic density between junctions
- 🗓️ Date-based dataset selection
- ✅ Input validation and exception handling
- 🖥️ Interactive GUI using Tkinter
